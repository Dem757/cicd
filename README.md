# CI/CD Lab: GitHub Actions

## Overview
This repository demonstrates a C-based CI/CD pipeline using GitHub Actions. It covers automated linting, unit testing, building, and deployment.

## Key Features
- Continuous Integration: Automated static analysis (splint) and functional testing.
- Continuous Deployment: Automated build process with downloadable artifacts.
- Status Monitoring: Real-time pipeline status badges in the README.

## Tools
- Language: C
- CI/CD Platform: GitHub Actions
